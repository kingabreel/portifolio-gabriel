Projetos

Aqui você pode ver um dos meus principais projetos, sejam eles acadêmicos, profissionais ou projetos pessoais.

Forum hub - API

API implementada para servir como um forum, utilizando spring boot, foi implementado funcionalidade de criação de usuários, tópicos e respostas a tópicos.

Biblioteca

API de um sistema de gerenciamento, busca e salvamento de livros. Faz o consumo de uma API externa para buscar livros e em seguida pode salvar os resultados em uma API interna.

Sistema de Eleição

Sistema de gerenciamento de eleição, com possibilidade de cadastro de candidados e adição de votos para os mesmos.

Sistema de comercio de cartas

Consumindo a API de rick e morty, foi implementado um sistema de venda e compra de cartas, utilizando angular.

Sonic's game

Jogo implementado em html, css e js. Objetos vem em direção ao personagem e o usuário deve pular sob os objetos, semelhante ao dino game no chrome.

Poke-api

Consumo da api dos pokemons e mostrando cada pokemon na página inicial, ao clicar em qualquer pokemon será mostrado suas informações.

Lista de tarefas

Utilizando angular, é possível adicionar tarefas com um título. Cada tarefa adicionada possui um ID específico, podendo também ser excluida ou marcada como concluida.

Jogo da cobrinha

Implementado funcionalidades como aumentar a dificuldade do jogo, modificar o tamanho do mapa e configurar a velocidade da cobra.
